RU

practice app to plot a graph by points
это консольная программа на Python для построения графиков по точкам. Она позволяет вводить координаты точек через консоль или из файла, визуализировать их с помощью различных библиотек (Matplotlib для статических графиков, Bokeh для интерактивных) и сохранять результаты в различных форматах.

Программа разработана в рамках учебной практики и может быть использована в образовательных целях, научных исследованиях и инженерных расчетах.
Операционная система: Windows 10/11, Linux, macOS

Python: версия 3.9 или выше

Оперативная память: от 512 МБ

Место на диске: от 100 МБ

Для установки необходимых библиотек следует выполнить команды:
pip install matplotlib numpy bokeh scipy
pip install black
pip install isort pre-commit

ENG

practice app to plot a graph by points
This is a Python console program for plotting graphs by points. It allows you to enter point coordinates through the console or from a file, visualize them using various libraries (Matplotlib for static graphs, Bokeh for interactive graphs), and save the results in various formats.

 The program was developed as part of a learning exercise and can be used for educational purposes, scientific research, and engineering calculations.
Operating system: Windows 10/11, Linux, macOS

Python: version 3.9 or higher

RAM: from 512 MB

Disk space: from 100 MB

To install the required libraries, run the following commands:
pip install matplotlib numpy bokeh scipy
pip install black
pip install isort pre-commit
